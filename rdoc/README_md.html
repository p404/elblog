    Elblog es un blog minimo hecho desde un rails engine, que usa las siguientes gemas:

    Bootsy Bootstrap-wysihtml5 CarrierWave.

    Instalación¶ ↑

    1. Agrega elblog a tu Gemfile: ruby gem 'elblog', git: 'https://github.com/p404/elblog'

    2. Instalar la gema: console bundle install

    3. Corre el generador: console rails generate elblog:install <target_mount_path>

    Nota: si no se usa <target_mount_path> se montara en la ruta “/elblog”

    1. Por ultimo instalar y correr las migraciones: console rake elblog:install:migrations rake db:migrate

    2. Si deseas personalizar las vistas ejecuta este comando console rails generate elblog:views

    Esta comando copiará las vistas show, edit, index, new y form dentro de la carpeta app/views de tu aplicación
